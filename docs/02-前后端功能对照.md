# WaiChat 前后端功能对照

## 页面与后端能力映射

| 前端页面/模块 | 关键前端文件 | 主要接口/通道 | 后端实现位置 | 功能说明 |
|---|---|---|---|---|
| 登录页 | `WaiChat-vue/src/views/Login.vue` | `POST /api/login` | `SecurityConfig` + `LoginSuccessHandler` | 用户鉴权并返回用户基础信息 |
| 注册页 | `WaiChat-vue/src/views/Register.vue` | `POST /api/register` | `LoginController.register` | 创建账号并加密密码 |
| 聊天主界面 | `WaiChat-vue/src/views/Chat.vue` | `GET /api/chat/getContactList` | `ChatController.getContactList` | 拉取联系人列表 |
| 聊天历史 | `WaiChat-vue/src/views/Chat.vue` | `GET /api/chat/history` | `ChatController.getChatHistory` | 拉取与目标用户聊天记录 |
| 发送消息 | `WaiChat-vue/src/views/Chat.vue` | `WebSocket /ws/{userId}` + `POST /api/chat/send` | `WebSocket.sendOneMessage` + `ChatController.sendMessage` | 实时消息发送与兜底 |
| 隐藏会话 | `WaiChat-vue/src/views/Chat.vue` | `POST /api/chat/removeHistory` | `ChatController.removeHistory` | 将会话标记为隐藏 |
| 恢复会话 | `WaiChat-vue/src/views/Chat.vue` | `POST /api/chat/recoverHistory` | `ChatController.recoverHistory` | 恢复被隐藏会话 |
| 搜索联系人 | `WaiChat-vue/src/views/Chat.vue` / `AddContactModal.vue` | `GET /api/user/search` | `UserController.searchUser` | 用户名/昵称/ID 检索 |
| 语音上传 | `WaiChat-vue/src/views/Chat.vue` | `POST /api/upload` | `FileController.uploadAudio` | 上传音频并返回可访问 URL |
| 语音识别（URL） | `WaiChat-vue/src/views/Chat.vue` | `POST /api/ai/voiceToText` | `AIController.voiceToText` | 通过音频 URL 转文字 |
| 语音识别（文件） | `WaiChat-vue/src/views/Chat.vue` | `POST /api/ai/audio/stt` | `AIController.transcribe` | 直接上传文件做语音转写 |
| 获取语言列表 | `WaiChat-vue/src/views/Chat.vue` | `GET /api/ai/languages` | `AIController.getLanguages` | 翻译目标语言选项 |
| 文本翻译 | `WaiChat-vue/src/views/Chat.vue` | `POST /api/ai/translate` | `AIController.translate` | 目标语言翻译 |
| 文本润色 | `WaiChat-vue/src/views/Chat.vue` | `POST /api/ai/polish` | `AIController.polish` | 商务/休闲风格润色 |
| 智能回复 | `WaiChat-vue/src/views/Chat.vue` | `POST /api/ai/smartReply` | `AIController.smartReply` | 根据上下文生成回复建议 |
| 聊天总结 | `WaiChat-vue/src/views/Chat.vue` | `POST /api/ai/summarize` | `AIController.summarize` | 自动总结历史聊天 |
| 聊天分析 | `WaiChat-vue/src/views/Chat.vue` | `POST /api/ai/analysis` | `AIController.analysis` | 关键词/情感/摘要分析 |

## 数据流对照（核心）

- **鉴权流**：`Login.vue` -> `/api/login` -> Spring Security 鉴权 -> 成功处理器返回用户信息。
- **实时消息流**：`Chat.vue` -> WebSocket 发送 -> `WebSocket.sendOneMessage` -> Redis 热数据 + 异步 MySQL -> 推送目标用户。
- **AI 能力流**：`Chat.vue` -> `/api/ai/*` -> `AIController` -> DashScope -> 标准 `Result` 返回。
- **文件流**：前端上传音频 -> `/api/upload` 存储至 `uploads/audio` -> 返回 URL -> 可继续调用语音识别接口。
