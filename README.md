## 毕业设计：AI多语言聊天辅助平台


本项目是一个基于 Spring Boot + Vue.js 实现的现代化实时聊天应用。它不仅仅是一个基础的通讯工具，更集成了强大的大语言模型（LLM）能力，旨在通过 AI 提高用户的沟通效率、优化表达，并提供深度的聊天数据分析，使其成为一款智能化的协作辅助工具。

-----

## ✨ 核心功能亮点

本项目超越传统聊天应用，深度集成 AI，提供以下核心亮点：

### 1\. 深度 AI 辅助工具

* **智能润色与风格转换 (AI Polishing):** 实时将输入文本一键转换为商务、休闲、或更专业的风格，确保表达得体。
* **多轮对话摘要 (Chat Summarization):** 一键总结与联系人最近的聊天记录，快速提取核心议题、关键决策和待办事项。
* **智能回复推荐 (Smart Reply):** 根据历史对话内容，AI 实时推荐 3-5 个相关性高的回复选项，提升输入效率。

### 2\. 国际化与多语言支持

* **实时翻译:** 支持用户发送和接收消息的自动或手动翻译，打破语言障碍，支持多种主流语言。
* **语言检测:** 自动识别消息语言，并根据用户的设置提供翻译。

### 3\. **数据可视化与关系洞察 (核心亮点)**

* **AI 聊天数据看板 (Analytics Dashboard):**
    * **关键词分析:** 通过柱状图展示高频热词，洞察聊天的核心话题。
    * **情感波动趋势:** 实时分析最近消息的情感倾向（积极/中性/消极），帮助用户了解沟通氛围。
    * **关系总结:** AI 基于历史数据生成趣味性的关系总结与评价。

## 💻 技术栈

### ⚙️ 后端技术 (WaiChat-master)

| 模块 | 技术 | 描述 |
| :--: | :--: | :--- |
| **核心框架** | Spring Boot | 快速开发、配置简单的 Java Web 应用。 |
| **实时通信** | WebSocket | 实现全双工通信，确保消息实时推送。 |
| **AI 集成** | 通义千问（Qwen） API 调用 | 统一的消息处理工具，负责与 LLM 服务接口对接。 |
| **数据持久化** | MyBatis Plus | 简化数据库操作（CRUD）。 |
| **数据分析** | Jackson / Fastjson | 处理和解析 AI 返回的 JSON 结构化数据。 |

### 🖥️ 前端技术 (WaiChat-vue)

| 模块 | 技术         | 描述 |
| :--- |:-----------| :--- |
| **核心框架** | Vue.js 3   | 采用组件化开发，构建用户界面。 |
| **状态管理** | Vuex/Pinia | 集中管理应用状态（如联系人列表、消息历史）。 |
| **可视化** | ECharts    | 用于渲染聊天数据分析看板中的图表（关键词、情感趋势）。 |

## 🔧 环境与部署

### 1\. 开发环境要求

* **JDK:** 17+
* **Maven:** 3.6+
* **Node.js:** 20+
* **数据库:** MySQL

### 2\. 部署步骤

#### A. 后端部署 (Spring Boot)

1.  **克隆代码:**
    ```bash
    git clone https://github.com/Caaati/WaiChat-master.git
    ```
2.  **配置数据库:** 修改`application.properties`文件，配置数据库连接信息。
3.  **通过springboot启动后端服务:**
    ```
    运行 WaiChat-master\src\main\java\com\zafu\waichat\WaiChatApplication.java
    ```
4.  **通过jar包启动后端服务:**
    ```bash
    # 使用 Maven 编译打包
    mvn clean package
    # 运行jar包
    java -jar target/waichat-master.jar
    ```

#### B. 前端部署 (Vue.js)

1.  **克隆代码:**
    ```bash
    git clone https://github.com/Caaati/WaiChat-vue.git
    ```
2.  **安装依赖:**
    ```bash
    npm install
    ```
3.  **启动前端服务:**
    ```bash
    npm run dev
    # 前端页面：http://localhost:5173/
    ```
